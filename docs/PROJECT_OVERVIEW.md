# Project Overview

## Problema

Los estudiantes de estructuras discretas tienen dificultades para visualizar conceptos abstractos de lógica proposicional, teoría de conjuntos y grafos. La falta de herramientas interactivas dificulta la comprensión profunda de estos temas fundamental es en ciencias de la computación.

## Solución

Una plataforma interactiva desarrollada en Angular que permite simulate, visualizar y practicar conceptos de estructuras discretas de manera dinámica. La aplicación ofrece módulos especializados para cada tema con retroalimentación inmediata y explicaciones generadas por IA.

## Alcance

### Qué incluye

- **Módulo de Lógica Proposicional**
  - Ingreso de proposiciones y conectores lógicos
  - Generación automática de tablas de verdad
  - Evaluación de expresiones lógicas

- **Módulo de Conjuntos**
  - Creación de conjuntos por extensión y comprensión
  - Operaciones: Unión, Intersección, Diferencia, Complemento
  - Visualización diagramas de Venn

- **Módulo de Grafos**
  - Creación de nodos y aristas
  - Representación mediante lista de adyacencia
  - Algoritmos: BFS (Breadth-First Search) y DFS (Depth-First Search)
  - Visualización gráfica del recorrido

- **Integración de IA**
  - Explicación automática de resultados
  - Generación de ejercicios
  - Interpretación de expresiones en lenguaje natural

### Qué NO incluye

- Sistema de autenticación de usuarios
- Base de datos persistente (solo sesión actual)
- Evaluación formal de evidencia lógica
- Teoremas de demostración automática
- Exportación a PDF/DOCX
